Português
Ganhe Milhares de Curtidas (Likes) com este super plugin. Exiba o botão de download somente após o visitante clicar no botão Curtir. 

English
Grown Up your Likes with this viral like plugin. Display download button only after user like some site page or Facebook fanpage.


== Description ==

**Apresentação**

Este plugin permite você ocultar o botão de download, obrigando o visitante do teu site a curtir sua página no Facebook através do botão Curtir que é exibido na mesma página. Depois que o visitante curte a página o botão de download é exibido automaticamente.

Você pode definir qual imagem de botão de download você quer usar no página de configuração do plugin. 

Um ponto forte deste plugin é que ele trabalha com shortcode, ou seja, você pode em cada página web de seu site ter um botão de like para sites ou páginas do facebook diferentes, que após curtidas irão revelar o botão de download para arquivos de download também diferentes, conforme você definiu. 

https://www.youtube.com/watch?v=7WWvGOgbJHU

**Sobre**

O plugin Baixar Após Curtir no Facebook foi escrito por Anderson Makiyama do [Como Ganhar Dinheiro na Internet](http://ganhardinheiroblog.net)


== Installation ==

Para instalar e usar o plugin é muito simples:

1- Envie a pasta do plugin para dentro da pasta de plugins, que geralmente é /wp-content/plugins/

2- Faça login no painel de gerenciamento do teu blog WP, vá na seção plugin e ative o Plugin Baixar apos Curtir no Facebook

3- Agora, no lado esquerdo do painel do blog, uma nova opção chamada "Baixar apos Curtir no Facebook" aparecerá, click sobre esta nova opção para ir para a página de opções.

4- Na página de opções, informe o url da imagem do botão de download e salve as alterações. Note que você também pode usar a imagem padrão, caso deseje. 


Para usar, em qualquer página do blog basta colocar as tags: [aposcurtir]url_to_like, download_url[/aposcurtir]

Substitua url_to_like pela url do site ou página do facebook que é para ser curtida e substitua download_url pelo url do download.
